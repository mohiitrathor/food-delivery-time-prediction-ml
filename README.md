# Food Delivery Time Prediction using Machine Learning

This project predicts whether a food delivery will be **Fast or Delayed** using multiple machine learning classification algorithms.

## Project Overview
Food delivery platforms rely heavily on accurate delivery time estimation. In this project, machine learning models are used to classify deliveries as **Fast (0)** or **Delayed (1)** based on various features.

## Techniques Used
- Data Preprocessing
- Feature Engineering
- Haversine Distance Calculation
- Feature Scaling
- Machine Learning Classification

## Models Implemented
1. Naive Bayes (Gaussian)
2. K-Nearest Neighbors (KNN)
3. Decision Tree

## Workflow
1. Data Cleaning & Preprocessing
2. Feature Engineering
3. Train-Test Split
4. Model Training
5. Performance Evaluation

## Evaluation Metrics
- Confusion Matrix
- ROC Curve
- Model Comparison

## Key Insights
- Distance between restaurant and customer strongly impacts delivery delay.
- Tree-based models provide interpretability through feature importance.
- Hyperparameter tuning improves KNN performance.
